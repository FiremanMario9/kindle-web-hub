# Kindle Web Hub
- A website compatable with the limited Kindle Web Browser as a hub for websites, online games, search(ing) and more.

# Homebrew
- In the Homebrew folder, there will be homebrew applications for KUAL, that have been renamed to **.zip.txt.
- Go to the linked website on your kindle, and go to the homebrew website. [https://firemanmario9.github.io/kindle-web-hub/homebrew/index.html]
- Click on the homebrew app you want.
- The web browser will ask you if you want to download the file. Click "yes".
- Let it download.
- Open KUAL.
- Open Homebrew Browser.
- Open HBB: Install apps.

# Troubleshooting:
Q: Help! I don't see "HBB: Install apps"!
A: Make sure you downloaded the extractor program into your "mnt/us/extensions" folder.
Q: The app dosen't show up in KUAL!
A: Make sure you have Internet, and that the file shows up in your library as **.zip.txt. Check the Logs. If all else fails, plug your kindle into your PC and check to see that it extracted to "mnt/us/extensions". If that dosen't work, it's a problem with your KUAL installation.
Q: Can I add my own homebrew app?
A: Sure! Make sure it is zipped and the program is not in an "extensions" folder in the zip. Rename the zip to .zip.txt and email me the file. If it is too large, share a google drive link with me, or use mediafire, or just upload it to Github and send me the link.
